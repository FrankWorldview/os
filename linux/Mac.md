# Installing Ubuntu on Mac

The three solutions might work on Mac + Ubuntu.

Solution 1: When you download Virtual Box, choose "macOS / Apple Silicon hosts", instead of "macOS / Intel hosts".

Solution 2: Please see the [article](https://hackmd.io/@cpt/MacBook_VBox_Ubuntu_for_ARM).

Solution 3: Download the [Ubuntu VirtualBox image](https://www.linuxvmimages.com/images/ubuntu-2404/) and then import it into your VirtualBox.

Solution 4: Please see the [article](https://hackmd.io/@wyping/SJ6LpLr-a).
